# Ritual Penang - Audio Files

This directory should contain the following audio files for the interactive experience:

## Required Audio Files:

1. **ritual-sound.mp3**
   - Triggered when clicking on the logo model
   - Suggestion: A ritual bell or singing bowl sound

2. **fullmoon-sound.mp3**
   - Plays when Full Moon Mode is activated (M key)
   - Suggestion: Atmospheric moonlight ambience, soft drones

3. **portal-sound.mp3**
   - Plays during teleportation effects
   - Suggestion: Whoosh or ethereal transition sound

4. **easteregg-sound.mp3**
   - Plays when typing "RITUAL" (easter egg)
   - Suggestion: Mystical chime or deep resonance

5. **ambient.mp3**
   - Loops continuously as background atmosphere
   - Suggestion: Subtle ritual ambience, drone, or field recordings

## File Format:
- All files should be in MP3 format
- Recommended: Mono or stereo, 128-192 kbps
- Keep file sizes reasonable for web loading

## Where to Find Sounds:
- [Freesound.org](https://freesound.org) - Free sound effects
- [Epidemic Sound](https://epidemicsound.com) - Licensed music
- Record your own ritual sounds and ambiences

## Note:
The website will work without these files, but audio features will be silent. Make sure to add these files before deploying the website.
